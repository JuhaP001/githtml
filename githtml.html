Git Perusteet ja Keskeiset Komennot

Git on hajautettu versionhallintajärjestelmä, joka on suunniteltu seuraamaan muutoksia lähdekoodissa ohjelmistokehityksen aikana. Tässä on joitakin keskeisiä Git-komentoja ja niiden esimerkkejä.

Git Repositorion Luominen

Uuden Git-repositorion luominen:

git init

Repositorion Kloonaaminen

Olemassa olevan repositorion kloonaaminen:

git clone <repository_url>

Muutosten Seuraaminen

Tiedostojen lisääminen seurantaan:

git add <tiedosto>

Kaikkien tiedostojen lisääminen seurantaan:

git add .

Muutosten Sitominen

Muutosten sitominen (commit):

git commit -m "Viesti muutoksista"

Muutosten Työntäminen

Muutosten työntäminen etärepositorioon:

git push origin <branch_name>

Muutosten Hakeminen

Muutosten hakeminen etärepositoriosta:

git pull origin <branch_name>

Haara (Branch)

Uuden haaran luominen:

git branch <branch_name>

Haaran vaihtaminen:

git checkout <branch_name>

Haara Merging

Haaran yhdistäminen nykyiseen haaraan:

git merge <branch_name>

Git Status

Repositorion tilan tarkistaminen:

git status

Git Log

Commit-historian tarkistaminen:

git log
